Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is case sensitive.
  • HTML uses opening and closing tabs.
  • It starts with doctype, then the lang choice, then the head and meta, title, close head, and then body

CSS

  • A Selector defines the element or attributes to which the rules or declarations will apply
  • Declarations contain 2 components: the CSS property we want to apply and the value of the property
  • Margin Property: indicates how much space we want around the outside of an elemetn. It functions as a buffer.
  • Padding Property: adds space around the content inside an element. More padding means bigger element.
  • Using a * we declare that ALL of the elements on the page have the style properties as allotted.
  • DRY aka Do Not Repeat Yourself: coding principle that encourages devs to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained. DRY can greatly reduce the overall size and complexity of a codebase.
  • the img selector assigns property values to any image element in the markup
  • The link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current doc and the linked doc.
  • The href attribute specifies the location (URL) of the external resource.

Git

  • git checkout -b branch-name: creates a new branch and switches to it
